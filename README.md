# 🇩🇴 Trabajo Fácil — Empleos en República Dominicana

Plataforma de empleos simple y rápida para República Dominicana.

## 🌐 Ver en vivo

[https://wascarelia8-lang.github.io/trabajofacil-FINAL.html/](https://wascarelia8-lang.github.io/trabajofacil-FINAL.html/)

## ✨ Funciones

- Buscar empleos por ciudad, categoría y modalidad
- Aplicar por WhatsApp, correo o formulario
- Panel para empresas — publicar y gestionar vacantes
- Ver candidatos y exportar CSV
- Panel Admin completo
- Diseño dark mode, mobile-first

## 🛠️ Tecnologías

- HTML + CSS + JavaScript (un solo archivo)
- Supabase — base de datos y autenticación
- GitHub Pages — hosting gratuito

## 👤 Admin

Para hacer un usuario admin, ejecutar en Supabase SQL Editor:

```sql
SELECT make_admin('tu@email.com');
```
